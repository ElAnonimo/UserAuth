# Описание
Система аутентификации пользователя, основанная на Sails. Система использует полный MEAN стек.

### Технологии
* SailsDB
* MongoDB
* Nodemachine
* Gravatar
* CSS
* Bootstrap
* Grunt

### Установка модулей
`npm install`

### Запуск приложения
`sails lift`
В меню выберите опцию 2.

Приложение будет доступно по адресу `http://localhost:1337`
